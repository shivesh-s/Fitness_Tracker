# Fitness Tracker

- A simple python program which provides the user with an interface to store their workout history to track their progress.
- the data is stored in local MySQL, using mysql-connector

## Development Progress

## Installation and Usage

> Dependencies:
> - mysql-connector-python

1. Clone the repository
<pre>git clone https://github.com/shivesh-s/Fitness_Tracker.git</pre>

2. Install the required dependencies
<pre>pip install -r requirements.txt</pre>

3. Run the main.py file using python
<pre>cd Fitness_Tracker
python main.py</pre>
