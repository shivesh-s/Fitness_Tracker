# Fitness Tracker

- A simple python program which provides the user with an interface to store their workout history to track their progress.
- the data is stored in local MySQL, using mysql-connector

## Installation and Usage
---
### Dependencies:
- mysql-connector-python

1. Clone the repo
`git clone https://github.com/shivesh-s/Fitness_Tracker.git`

2. install the required dependencies
```pip install -r requirements.txt```

3. run the main.py file using python
```cd Fitness_Tracker```
```python main.py```
